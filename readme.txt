Salesmate Integration for Contact Form 7 is a minimal plugin that creates a Salesamate Contacts, Company and Deal submitted for you're done.

== Description ==
This is a child plugin(Add-on) of Contact form 7 plugin. It Allows you to create contacts along with company as well as deal by submitting the Contact Form directly in Salesmate CRM.

= Features =
* Create Contact.
* Create Deal.
* Create Contact and Deal both along with company.
* Dynamic salesmate field mapping wiht contact form7's field.

== Installation ==

This section describes how to install the plugin and get it working.

NOTE:- To add this plugin you must have Contact Form 7(parent plugin) installed in your site.

1. Upload .zip file from "Add New" at admin site in "Plugin" menu OR put unzip folder directly at location "/wp-content/plugins"
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Goto 'Contact > Salesmate Integration' and provide required credentials from your Salesmate CRM.
